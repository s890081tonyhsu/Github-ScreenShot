Github-ScreenShot
=================

What's this?
----------

It's a tools to get screenshot from the following site.
```ruby
    # account <= "your account here"
    "github.com/#{account}"
    "#{account}.github.io"
```
Requirement
----------
> bundle install  

'Ruby' , '>=2.0.0'  
'Selenium-WebDriver', '>= 2.43.0'
